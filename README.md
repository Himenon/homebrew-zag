# homebrew-zag

## Install

```bash
brew tap Himenon/zag
brew install zag
```

## Upgrade

To upgrade, it is recommended that you run the `brew upgrade` command.

```bash
brew upgrade zag
```

## Uninstall

If you want to uninstall Zag, run the following commands:

```bash
brew uninstall zag
brew untap Himenon/zag
```
